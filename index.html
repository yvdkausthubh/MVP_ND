About the Project

This project integrates Raspberry Pi to display advertisements on the back of autos based on the user's pincode. It also includes a shock sensor for enhanced functionality.

How It Works

The Raspberry Pi 4B serves as the core operating system in our prototype, managing commands from the server to play ads on a display screen while providing real-time detection. Using a GSM module, it automatically captures GPS data, including the pin code of the auto-rickshaw's location. This data is sent to the server, which determines the relevant advertisements to display based on the location.

The system retrieves video files from a memory card, categorized into folders corresponding to specific pin codes. For example, if the auto-rickshaw enters pin code 500045, the Raspberry Pi accesses the "500045" folder and plays only the advertisements stored there.

The display screen, approximately 100 meters in size, is designed for high brightness, ensuring that the advertisements are visually appealing and have high resolution. This makes the ads attractive to drivers, passengers, and passersby. The ads can include QR codes or promotional details, such as discounts available at nearby stores. This allows viewers to scan the QR codes for more information or to redeem offers.

This MVP integrates the Raspberry Pi 4B with a mobile app or website to manage data and deliver ads seamlessly. The system's adaptability ensures that location-specific content is displayed, making it an engaging and impactful advertising medium.
